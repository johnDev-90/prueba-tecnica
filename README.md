# Prueba Técnica – Frontend

Este repositorio contiene el desarrollo del **frontend** correspondiente a una prueba técnica.

El enfoque del trabajo fue la construcción de la interfaz de usuario, la experiencia visual, y el consumo de una API REST.  
El backend **no fue desplegado** como parte de esta entrega.

---

## Descripción del proyecto

Aplicación web desarrollada con **Next.js** que permite visualizar y gestionar productos.

Incluye listado de productos, búsqueda, categorías, formularios y diseño responsivo, consumiendo datos desde una API externa configurada mediante variables de entorno.

---

##  Tecnologías utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS
- DaisyUI
- SweetAlert
- React Toastify

---

##  Funcionalidades

- Listado de productos
- Búsqueda de productos
- Categorías
- Formularios
- Diseño responsivo
- Consumo de API REST
- Notificaciones y alertas visuales

---

## ⚙️ Instalación y ejecución

### Requisitos

- npm

```bash

# Ejecutar el proyecto
